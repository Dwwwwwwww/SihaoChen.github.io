Scented Days: Exploring the Capacity of Smell Narratives

A game created to raise awareness of Smell Loss. Also A Paper presented on CHI Play 2024.

This project focuses on making a Serious Game that can be interactive on Smell. We created an apparatus that emits odor with the game using Arduino. Game made in Unity.
